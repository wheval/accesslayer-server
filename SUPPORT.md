# Support

## Getting help

- For backend bugs, use the bug report issue template.
- For scoped implementation work, use the maintainer task template.
- For setup and docs gaps, use the documentation template.

## Before opening an issue

- Read [CONTRIBUTING.md](./CONTRIBUTING.md).
- Review [docs/open-source/issue-backlog.md](./docs/open-source/issue-backlog.md).
- Confirm the work belongs in the server repo and not the client or contracts repo.

## Project boundaries

This repository covers API, metadata, auth, indexing, and gated off-chain access. Frontend UX belongs in `accesslayer-client`. On-chain pricing, supply, and fee rules belong in `accesslayer-contracts`.
